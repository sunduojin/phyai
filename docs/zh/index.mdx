# PhyAI 是什么

**PhyAI**（**Phy**sical **AI** Inference Framework，意为“物理 AI 推理框架”，读作 “phi”）是一个面向物理 AI 模型的高性能推理框架，支持 VLA、WAM 等模型及更多扩展场景，可同时用于云端服务和端侧部署。
